### 上手指南

运行 mario.py 即可

```py3
cd <Your Directory>
python mario.py
```

###### 开发前的配置要求
1. 安装 python 环境（推荐安装 python 3.7）
2. 安装 paddle 2.0 及以上版本
3. 安装 opencv 4.5.3 及其以上版本
4. 安装 pip

###### **其他依赖安装**

进入项目当前目录，并使用 pip 安装其余依赖

```py3
cd <Your Directory>
pip install -r requirement.txt
```
